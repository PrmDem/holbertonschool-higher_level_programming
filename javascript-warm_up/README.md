# JavaScript - Warm Up
In order to get familiar with JavaScript, the following tasks will be JS scripting.<br/>

## Requirements
* Allowed editors: vi, vim, emacs
* All files will be interpreted on Ubuntu 20.04 LTS using `node` (version 14.x)
* All files should start with `#!/usr/bin/node` and end with a new line
* The code should be `semistandard` compliant (version 16.x.x). Rules of Standard + semicolons on top.
* All files must be executable
* File length will be tested using `wc`

First use of the console to print a most truthful statement.<br/>
