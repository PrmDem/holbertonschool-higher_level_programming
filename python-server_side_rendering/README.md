# Server-side Rendering
For more efficient and maintainable websites, and in order to make SEO (_**S**earch **E**ngine **O**ptimisation_) easier to attain, server-side rendering (_SSR_) is the way to go.<br/>
In this project, we'll learn how to replace Client-side rendering with SSR using `Python`, `Flask`, and `Jinja`.<br/>
