Test cases for ``1-my_list`` module
======================

>>> print_sorted = __import__('1-my_list').MyList.print_sorted


Basic case use:

>>> print_sorted([3, 2, 4, 5, 1])
[1, 2, 3, 4, 5]


Only one number:

>>> print_sorted([17])

All the same number:

>>> print_sorted([2, 2, 2])
[2, 2, 2]


Empty list:

>>> print_sorted([])

Positive and negative numbers:

>>> print_sorted([-1, 5, 0, -3])
