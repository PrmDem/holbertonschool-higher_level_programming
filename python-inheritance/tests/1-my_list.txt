Test cases for ``1-my_list`` module
======================

Using ``print_sorted``
-------------------

Getting ``print_sorted`` from the ``1-my_list`` module:

>>> print_sorted = __import__('1-my_list').MyList.print_sorted


Basic case use:

>>> print_sorted([2, 7, 15, 4, 21])
[2, 4, 7, 15, 21]


Only one number:

>>> print_sorted([1])
[1]


All the same number:

>>> print_sorted([2, 2, 2])
[2, 2, 2]


List is empty:

>>> print_sorted([])
[]


Duplicate of a number:

>>> print_sorted([1, 4, 8, 1, 2])
[1, 1, 2, 4, 8]
